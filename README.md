# tela-google
Clone da Tela Inicial do Google
Este projeto é uma recriação da tela inicial do Google desenvolvida com HTML e CSS, com o objetivo de praticar conceitos de estruturação de páginas web, estilização, responsividade e organização de elementos na interface.

Funcionalidades
Logo centralizada.
Barra de pesquisa estilizada.
Layout semelhante ao da página inicial do Google.
Design responsivo para diferentes tamanhos de tela.

Tecnologias Utilizadas
HTML5
CSS3

Objetivo
Este projeto foi desenvolvido para aprimorar habilidades em desenvolvimento front-end, praticando posicionamento de elementos, flexbox, responsividade e reprodução de interfaces conhecidas.

Aprendizados
Durante o desenvolvimento foram trabalhados conceitos como:
Estrutura semântica com HTML.
Estilização com CSS.
Uso de Flexbox.
Manipulação de SVGs e ícones.
Responsividade para dispositivos móveis e desktops.

⚠️ Aviso: Este projeto foi criado apenas para fins educacionais e de estudo. O Google e sua identidade visual são marcas registradas da Google.
